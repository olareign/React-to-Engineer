## What this book teaches

Knowing React and engineering frontends are different skills. This book closes that gap: eight chapters, each building one real project in TypeScript, each carrying two altitudes — the main flow that builds working professional skills, and a **Level Up** section covering the tradeoffs, failure modes, and reasoning that senior engineers are paid for. The capstone is a production-grade conference companion app frontend, reviewed against a senior rubric.

Written for developers who know JavaScript and are new to React. A TypeScript primer in the front matter brings you up to speed; everything after it is typed.

Each chapter folder contains:

- `starter/` — the scaffold to begin the chapter from
- `final/` — a complete reference implementation
- `exercises/` — solutions to the chapter's mini exercises

## Requirements

- Node.js 20+
- npm (or pnpm/yarn)
- Projects use Next.js 15 with React 19 and TypeScript 5

## How to use this repo

1. Read the chapter first — the code makes sense in the order the book builds it.
2. Start from `starter/`, type the code yourself, and compare against `final/` when stuck.
3. Do the exercises before opening `exercises/`.
4. The capstone deliberately stops at the network boundary — its backend is Book 2.

## Continue the series

The capstone frontend in this book was designed to snap onto a real backend. **Book 2, *The Server Side of Next.js***, builds exactly that backend — same conference app, other side of the wire. The two capstones together are one complete product.
